# Agro Forte: Futuro Sustentável

## 🌱 Sobre o Projeto
O **Agro Forte: Futuro Sustentável** é um projeto educativo que busca promover a **sustentabilidade, a preservação ambiental e o desenvolvimento agrícola consciente**. O objetivo é conscientizar a sociedade sobre práticas sustentáveis e valorizar a agricultura moderna e responsável.

O site apresenta informações sobre o projeto, seus objetivos e formas de engajamento.

---

## 🎯 Objetivos
- Incentivar práticas agrícolas sustentáveis.  
- Promover a educação ambiental entre jovens e comunidades.  
- Valorizar o trabalho do produtor rural.  
- Criar consciência sobre a importância da agricultura para o futuro sustentável do planeta.
